# CashStream Admin Panel

Административная панель для управления бэкенд приложением CashStream.

## Структура

Админ-панель теперь интегрирована в бэкенд проект и находится в папке `admin/`.

## Установка

Из корня бэкенд проекта:

```bash
# Установить все зависимости (бэкенд + админ)
yarn install:all

# Или установить отдельно
yarn install
cd admin && yarn install
```

## Разработка

### Запуск обоих процессов одновременно (рекомендуется)

```bash
# Из корня бэкенд проекта
yarn dev:all
```

Это запустит:
- Бэкенд на `http://localhost:3000`
- Админ-панель на `http://localhost:5173`

### Запуск отдельно

```bash
# Бэкенд
yarn dev

# Админ-панель (в другом терминале)
yarn dev:admin
```

## Сборка

```bash
# Собрать всё (бэкенд + админ)
yarn build

# Или отдельно
yarn build:backend  # Только бэкенд
yarn build:admin    # Только админ
```

## Production

В production режиме админ-панель раздается как статические файлы через Express сервер:

```bash
# Собрать проект
yarn build

# Запустить production сервер
NODE_ENV=production yarn start
```

Админ-панель будет доступна на том же порту, что и API (по умолчанию 3000).

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev-сервер
- **React Router** - маршрутизация
- **Tailwind CSS** - стилизация
- **Axios** - HTTP клиент

## API Endpoints

Приложение использует следующие эндпоинты бэкенда:

- `GET /api/admin/users?page=1&limit=10` - получение списка пользователей с пагинацией
